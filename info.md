# Aria·R 项目背景与需求

## 项目概述
**Aria·R** 是一个私人读书网站，用于管理阅读记录、存储读书笔记和摘抄、追踪阅读进度，并提供跨书籍的知识关联与可视化呈现。
**核心理念**："轻量录入，智能处理" —— 用户只管简单记录，系统负责整理、关联、可视化、回顾。
**主要用途**：个人使用，偶尔分享给朋友。

## 名称释义
- **Aria（咏叹调）**：音乐中抒发情感的独唱段落，象征阅读是灵魂与文字的对话
- **R（Read）**：阅读，核心行为
- **Aria·R**：阅读即共鸣，每一次翻页都是一次灵魂的相遇

## UI/UX 设计规范
### 设计风格：现代文艺风
### 色彩系统
- 主背景：米白 #F8F6F0
- 卡片背景：浅灰 #F0F0F0
- 主文字：深灰 #2C2C2C
- 次要文字：中灰 #6B6B6B
- 主强调色：莫兰迪绿 #5B7E71
- 辅助强调：雾霾蓝 #6B8FAD
- 点缀色：暖棕 #A67C52
- 成功：柔和绿 #7BAE7F
- 警告：柔和橙 #D4A574
- 错误：柔和红 #C47C7C

### 字体
- 中文标题：霞鹜文楷
- 中文正文：思源宋体
- 英文标题：Lora
- 英文正文：Inter
- 代码/标签：JetBrains Mono

### 质感要素
1. **微妙阴影**：多层阴影，营造层次感
2. **细腻边框**：1px 淡色边框，精致度
3. **渐变与纹理**：纸张纹理背景，微妙渐变卡片
4. **圆角**：8px-12px，柔和不生硬
5. **留白**：充足的内外边距，呼吸感

### 动效与交互
1. **页面切换**：淡入淡出 + 轻微位移（200-300ms）
2. **卡片悬浮**：轻微上浮 + 阴影增强（ease-out）
3. **按钮点击**：缩放反馈（scale 0.98）+ 涟漪效果
4. **图谱交互**：节点hover高亮，点击展开详情面板
5. **加载状态**：骨架屏 + 渐进加载

### 布局
- 响应式设计：适配手机、平板、电脑
- 侧边栏导航：桌面端固定侧边栏，手机端底部Tab
- 内容区：最大宽度 1200px，居中
- 卡片网格：网格/瀑布流布局

### 首页设计
- 问候语："今天，读了什么？"
- 阅读进度概览
- 最近摘录回顾
- 正在读的书籍（行板中...）

## 功能模块
### 1. 书籍管理
- 添加新书：ISBN自动获取 + 手动录入
- 书籍封面、简介、出版信息
- 编辑/删除书籍

### 2. 阅读状态追踪
- 状态：序曲（想读）/ 行板（在读）/ 终章（已读完）
- 开始日期 / 完成日期
- 阅读进度（百分比或页数）
- 阅读时长统计

### 3. 摘录
- **原文**：书中划线的句子
- **感想**：可选，自由输入
- **录入方式**：快速打字 / 语音输入（Web Speech API自动转文字）
- **智能处理**：系统自动提取关键词、生成标签建议

### 4. 书评与评分
- 评分：1-5星 / 10分制
- 短评（一句话）
- 长评（自由撰写）

### 5. 标签与分类
- 手动标签：自定义标签
- 自动标签：AI根据内容自动生成建议
- 分类维度：主题、类型、年份、情绪、自定义维度

### 6. 角色管理（小说类）
- 角色信息：姓名、简介、性格、背景
- 角色头像/代表色
- 跨书籍角色关联

### 7. 共鸣图谱
- **单本书关系图**：书中角色关系
- **跨书籍关系图**：不同书籍之间的知识关联
- **交互**：
  - 点击节点 → 显示详情卡片
  - 缩放、拖拽、聚焦
  - 按关系类型筛选
  - 节点高亮与路径追踪

### 8. 搜索功能
- 全文搜索（笔记、摘抄、书评）
- 多维度筛选（标签、评分、状态、年份、作者）
- 搜索结果高亮
- 搜索历史

### 9. 知识关联
- 自动发现不同书籍之间的共同主题/概念
- 相似书籍推荐
- 关联摘抄聚类

### 10. 回响（回顾）
- 每日/每周推送过去的摘录
- "一年前的今天你在读什么"
- 随机回顾

### 11. 数据导入/导出
- 导入：豆瓣、CSV、JSON
- 导出：Markdown、PDF、JSON备份

## 数据模型
- Book: id, title, author, isbn, cover, description, publishedYear, publisher, pageCount, status, startDate, finishDate, progress, rating, review, tags[], characters[], excerpts[], notes[], createdAt, updatedAt
- Excerpt: id, bookId, content, thought, pageNumber, chapter, tags[], characters[], createdAt
- Character: id, name, description, personality, avatar, color, books[], relationships[]
- Tag: id, name, color, count
- KnowledgeLink: id, sourceBookId, targetBookId, sourceExcerptId, targetExcerptId, type, strength

## 技术栈偏好
- 前端：React 19 + TypeScript + Tailwind CSS + shadcn/ui
- 图谱：D3.js / ECharts / vis.js / React Flow
- 动画：Framer Motion
- 路由：BrowserRouter
- 数据存储：localStorage（Phase 1，后期迁移到后端）
